# Word-Guess-Game

1. When the page loads the user will be prompted with a word and can begin guessing by pressing any key. 
2. If the user guesses a correct letter it will show both under the "current word" and under letters already guessed 
3. If the user wins. it will tally the score and show the user a gif. Same things goes for a loss. 
4. If the user wants a hint, they can press the hint button for a statement related to the answer.
